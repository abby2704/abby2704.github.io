## 📂 Proyectos Destacados

### 1. Sistema de Recomendación - Plan Ceibal 🎓
Desarrollo de un modelo híbrido para personalizar la oferta de formación docente en Uruguay.
* **Problema:** Sobrecarga de opciones educativas; necesidad de optimizar las trayectorias formativas.
* **Solución:** Implementación de un sistema de recomendación utilizando **LightFM**, integrando interacciones usuario-item y metadatos sociodemográficos.
* **Enlace:** [Ver Notebook del Proyecto](PROYECTO_CEIBAL.ipynb)

### 2. Predicción de Clientes Potenciales (EdTech) 📈
Modelo de clasificación para identificar leads de alta conversión en una plataforma educativa.
* **Problema:** Identificar usuarios con mayor probabilidad de adquirir cursos pagos.
* **Técnicas:** Análisis Exploratorio de Datos (EDA) profundo y entrenamiento de modelos supervisados de clasificación.
* **Enlace:** [Ver Notebook de Predicción](Learner_Notebook_Full_Code_Version_Potential_Customers_Prediction%20(2).ipynb)

### 3. Segmentación de Clientes (Aprendizaje No Supervisado) 👥
Análisis de la heterogeneidad de una base de clientes para optimizar estrategias de marketing.
* **Problema:** Ineficiencia en campañas genéricas que no capturan las necesidades específicas de distintos perfiles de compradores.
* **Metodología:** Preprocesamiento de datos, Ingeniería de Características y Reducción de Dimensionalidad mediante **PCA**. Aplicación de **K-Means Clustering** para la definición de segmentos.
* **Resultados:** Identificación de perfiles clave (ej: "Clientes de alto valor", "Cazadores de ofertas") basados en ingresos, hábitos de gasto y respuesta a campañas.
* **Enlace:** [Ver Notebook de Segmentación](Customer_Segmentation_Notebook.ipynb)
